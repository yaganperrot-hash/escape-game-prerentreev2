⚠️ CODE CORROMPU

🎯 Mission
Corrigez le code HTML ci-dessous pour faire apparaître le bouton "Connectez-vous" et permettre aux étudiants de se connecter. Le bouton est présent dans le code mais... il est caché !

  • Le bouton devrait apparaître dans l'aperçu à droite
